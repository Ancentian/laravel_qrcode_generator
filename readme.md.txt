1. Install a New or Existing Project
2. Create Model, Controllers and Migrations. 
3. Create Database and add it to the .env file 
4. Create a Page with a create item form e.g Create product Form
5. Create an Index Page to show list of Products saved.
6. Create store methods to stored the products to database
7. In store mmethod use mt_rand to assign the item saved a random code 
	eg. $number = mt_rand(100000000, 9999999999);
8. Install the Milon package: https://github.com/milon/barcode assist in generating the barcodes.
9. Use the guide provided by Milon.
